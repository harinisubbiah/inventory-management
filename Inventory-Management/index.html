// 03 — Background

From a C console program to a real product.

The original version of Manifest ran in a terminal: a C program with a struct array, a printf-driven menu, and seven operations for keeping a stockroom honest. This rebuild keeps that same logic — the same checks, the same arithmetic — and gives it an interface a stockroom manager would actually want to open.

Every screen here is hand-built with HTML, CSS, and JavaScript. No frameworks, no UI kit — so every interaction, animation, and layout decision is one made on purpose.

  • Inventory logic ported directly from the original C functions
  • Zero dependencies — vanilla HTML5, CSS3, and ES6 JavaScript only
  • Persists to local storage, so the dashboard survives a refresh
inventory.c → inventory.js
